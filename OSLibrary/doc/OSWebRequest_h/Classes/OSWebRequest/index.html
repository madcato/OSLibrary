cancelRequest

Cancel the current request. 

-(void)cancelRequest;
Discussion

It's important to call this method before destroying the WebRequest object. This is the way to avoid that the delegate object receive method invocations when not expected.

download:withDelegate:

Send a GET petition to a web server. 

-(void)download:(NSString*)url withDelegate:(id<OSWebRequestDelegate>)delegate;
Parameters
url

URL to download

delegate

Delegate object that will receive notification. This object must implement the OSWebRequestDelegate protocol.

Discussion

Use this method to download any page or file in a HTTP server.
